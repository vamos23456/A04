# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/85b6254e-9c9b-418f-b4ae-61a3f12515d0

## Run Locally

Prerequisites:
- Node.js
- Python 3.10+

1. Install frontend dependencies:
   `npm install`
2. Configure env:
   copy `.env.example` to `.env.local`, fill `OPENAI_COMPAT_API_KEY`, and keep the default Kimi-compatible settings unless you want to use another OpenAI-compatible provider
3. Start Python export backend:
   `cd backend/python_export && python3 -m venv .venv && source .venv/bin/activate && pip install -r requirements.txt && uvicorn app:app --reload --host 127.0.0.1 --port 8000`
4. Start frontend:
   `npm run dev`

Now DOCX/PPTX export is generated by Python backend (`python-docx`, `python-pptx`).
