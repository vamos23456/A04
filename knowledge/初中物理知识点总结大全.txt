
初中物理知识点总结大全
一、声现象
1. 声音的产生与传播
声音由物体的振动产生，振动停止，发声停止。
声音的传播需要介质（固体、液体、气体），真空不能传声。
声速：固体中最快，液体次之，气体最慢；15℃空气中声速为340m/s。
2. 声音的特性
音调：由频率决定，频率越高，音调越高（频率：物体每秒振动的次数，单位Hz）。
响度：由振幅决定，振幅越大，响度越大；还与距离发声体的远近有关。
音色：由发声体的材料、结构决定，是辨别不同发声体的依据。
3. 噪声的危害和控制
噪声：从物理角度，指发声体做无规则振动发出的声音；从环保角度，指妨碍人们正常休息、学习、工作的声音。
噪声的控制：①在声源处减弱（如关音箱、装消声器）；②在传播过程中减弱（如植树、装隔音板）；③在人耳处减弱（如戴耳塞、耳罩）。
4. 声的利用
传递信息：如回声定位（蝙蝠、声呐）、B超、听诊器。
传递能量：如超声波清洗眼镜、击碎体内结石。

二、光现象
1. 光的直线传播
光源：能够自行发光的物体（分为自然光源、人造光源）。
光在同种均匀介质中沿直线传播，实例：影子、日食、月食、小孔成像（成倒立的实像）。
光速：真空中光速为3×10⁸m/s，光在气体中速度略小于真空中，液体、固体中依次减慢。
2. 光的反射
光的反射定律：反射光线、入射光线和法线在同一平面内；反射光线、入射光线分居法线两侧；反射角等于入射角（光路可逆）。
反射类型：①镜面反射（光滑表面，如镜子，反射光集中）；②漫反射（粗糙表面，如墙面，反射光发散），二者均遵循反射定律。
平面镜成像特点：成正立、等大的虚像；像与物到镜面的距离相等；像与物的连线与镜面垂直（像与物关于镜面对称）。
3. 光的折射
光的折射规律：光从一种介质斜射入另一种介质时，传播方向发生偏折；折射光线、入射光线和法线在同一平面内，分居法线两侧；光从空气斜射入水/玻璃时，折射角小于入射角，反之则大于（光路可逆，垂直入射时传播方向不变）。
折射实例：筷子在水中“弯折”、池水变浅、海市蜃楼、凸透镜成像。
4. 光的色散与看不见的光
色散：太阳光通过三棱镜分解为红、橙、黄、绿、蓝、靛、紫七色光，说明白光是复色光。
色光的三原色：红、绿、蓝；颜料的三原色：红、黄、蓝。
看不见的光：①红外线（热效应强，应用：遥控器、红外线测温仪、夜视仪）；②紫外线（化学效应强，能使荧光物质发光，应用：验钞机；适量照射促进钙吸收，过量有害）。
5. 透镜及其应用
（1）透镜的基本概念
凸透镜：中间厚、边缘薄，对光有会聚作用；凹透镜：中间薄、边缘厚，对光有发散作用。
焦点（F）：平行于主光轴的光经凸透镜会聚的点；焦距（f）：焦点到光心的距离。
（2）凸透镜成像规律及应用
物距u>2f，像距f<v<2f，倒立、缩小的实像，应用照相机；
物距u=2f，像距v=2f，倒立、等大的实像，应用测焦距；
物距f<u<2f，像距v>2f，倒立、放大的实像，应用投影仪、幻灯机；
物距u=f，不成像，应用探照灯；
物距u<f，像距v>u，正立、放大的虚像，应用放大镜。
（3）眼睛和眼镜
眼睛：晶状体相当于凸透镜，视网膜相当于光屏，成倒立、缩小的实像。
近视眼：晶状体太厚，折光能力太强，像成在视网膜前方，用凹透镜矫正。
远视眼：晶状体太薄，折光能力太弱，像成在视网膜后方，用凸透镜矫正。
显微镜：由目镜和物镜组成，均为凸透镜，物镜成倒立、放大的实像，目镜成正立、放大的虚像。
望远镜：由目镜和物镜组成，物镜成倒立、缩小的实像，目镜成正立、放大的虚像。

三、物态变化
1. 温度与温度计
温度：表示物体冷热程度的物理量，单位摄氏度（℃）。
温度计原理：利用液体热胀冷缩的性质（常用液体：水银、酒精、煤油）。
摄氏温度规定：标准大气压下，冰水混合物的温度为0℃，沸水的温度为100℃。
温度计使用：①玻璃泡全部浸入被测液体，不接触容器底和壁；②待示数稳定后再读数；③读数时玻璃泡继续留在液体中，视线与液柱上表面相平。
2. 熔化和凝固
熔化：物质从固态变为液态的过程，吸热；凝固：物质从液态变为固态的过程，放热。
晶体和非晶体的区别：晶体有固定的熔点（凝固点），熔化（凝固）时温度不变，如冰、海波、各种金属；非晶体无固定熔点（凝固点），熔化（凝固）时温度持续变化，如蜡、松香、玻璃、沥青。
晶体熔化条件：①达到熔点；②继续吸热；晶体凝固条件：①达到凝固点；②继续放热。
3. 汽化和液化
汽化：物质从液态变为气态的过程，吸热；液化：物质从气态变为液态的过程，放热。
汽化的两种方式：
①蒸发：在任何温度下都能发生，只在液体表面进行的缓慢汽化现象，影响因素：液体温度、表面积、表面空气流速。
②沸腾：在一定温度（沸点）下，在液体表面和内部同时发生的剧烈汽化现象；标准大气压下，水的沸点为100℃；沸腾条件：①达到沸点；②继续吸热（温度不变）。
液化的两种方法：①降低温度；②压缩体积（如液化石油气、打火机内的燃气）。
4. 升华和凝华
升华：物质从固态直接变为气态的过程，吸热，实例：干冰升华、樟脑丸变小、冰冻的衣服变干。
凝华：物质从气态直接变为固态的过程，放热，实例：霜、雪、雾凇、窗玻璃内侧的冰花。
5. 水循环
自然界的水通过熔化、凝固、汽化、液化、升华、凝华等物态变化，实现水的循环，伴随能量的转移。

四、电流和电路
1. 电荷
两种电荷：正电荷（用丝绸摩擦过的玻璃棒带正电）、负电荷（用毛皮摩擦过的橡胶棒带负电）。
电荷间的相互作用：同种电荷相互排斥，异种电荷相互吸引。
验电器：检验物体是否带电的仪器，原理：同种电荷相互排斥。
电荷量：电荷的多少，单位库仑（C）；元电荷e=1.6×10⁻¹⁹C。
摩擦起电：实质是电子的转移（失去电子带正电，得到电子带负电），不是创造了电荷。
2. 电流和电路
电流：电荷的定向移动形成电流，方向规定为正电荷定向移动的方向（与负电荷定向移动方向相反），金属导体中电流由自由电子的定向移动形成。
电流的单位：安培（A），常用单位：毫安（mA）、微安（μA），1A=10³mA=10⁶μA。
电路的基本组成：电源（提供电能）、用电器（消耗电能）、开关（控制电路）、导线（输送电能）。
电路的三种状态：①通路（处处连通，有电流）；②断路（某处断开，无电流）；③短路（电源两极直接相连，电流过大，烧坏电源，禁止）。
电路图：用符号表示电路连接的图（记住电源、开关、灯泡、电阻、电流表、电压表等符号）。
3. 串联和并联
（1）串联电路
连接：用电器依次连接，只有一条电流路径。
特点：①电流处处相等（I=I₁=I₂）；②开关控制整个电路，位置改变，作用不变；③一个用电器断路，其他用电器都不工作。
（2）并联电路
连接：用电器并列连接，有两条或多条电流路径。
特点：①干路电流等于各支路电流之和（I=I₁+I₂）；②各支路用电器独立工作，互不影响；③干路开关控制整个电路，支路开关只控制所在支路。
4. 电流的测量
仪器：电流表（符号A），有两个量程：0~0.6A（分度值0.02A）、0~3A（分度值0.1A）。
电流表使用规则：①串联在电路中；②电流从正接线柱流入，负接线柱流出；③不能超过量程；④绝对不允许直接接在电源两极上。
5. 串、并联电路电流的规律
串联电路：电流处处相等，即I = I₁ = I₂ = … = Iₙ。
并联电路：干路电流等于各支路电流之和，即I = I₁ + I₂ + … + Iₙ。

五、电压 电阻
1. 电压
作用：使电路中形成电流，电源是提供电压的装置。
单位：伏特（V），常用单位：千伏（kV）、毫伏（mV），1kV=10³V=10⁶mV。
常见电压值：一节干电池1.5V，一节蓄电池2V，家庭电路220V，人体安全电压不高于36V。
2. 电压的测量
仪器：电压表（符号V），有两个量程：0~3V（分度值0.1V）、0~15V（分度值0.5V）。
电压表使用规则：①并联在被测用电器两端；②电流从正接线柱流入，负接线柱流出；③不能超过量程；④可以直接接在电源两极上，测量电源电压。
3. 串、并联电路电压的规律
串联电路：总电压等于各部分电路两端电压之和，即U = U₁ + U₂ + … + Uₙ。
并联电路：各支路两端电压相等，且等于电源电压，即U = U₁ = U₂ = … = Uₙ。
4. 电阻
定义：导体对电流的阻碍作用，符号R，单位欧姆（Ω），常用单位：千欧（kΩ）、兆欧（MΩ），1MΩ=10³kΩ=10⁶Ω。
影响电阻大小的因素：导体的材料、长度、横截面积（材料相同，长度越长、横截面积越小，电阻越大）；还与温度有关（金属导体温度越高，电阻越大，灯丝电阻随温度升高明显增大）。
定值电阻：阻值固定不变的电阻；可变电阻：滑动变阻器、电阻箱，能改变接入电路的电阻。
5. 滑动变阻器
原理：通过改变接入电路中电阻丝的长度来改变电阻。
接线规则：一上一下（接上面两个接线柱，电阻为0；接下面两个接线柱，电阻最大，相当于定值电阻）。
作用：①改变电路中的电流和用电器两端的电压；②保护电路。
读数：电阻箱能直接读出接入的电阻值，滑动变阻器不能直接读数。

六、欧姆定律
1. 欧姆定律
内容：导体中的电流，跟导体两端的电压成正比，跟导体的电阻成反比。
公式：I = U/R（变形公式：U=IR、R=U/I）。
单位：I→安培（A），U→伏特（V），R→欧姆（Ω）。
注意：R是导体本身的性质，与U、I无关，不能说R与U成正比，与I成反比。
2. 电阻的串联和并联
串联电阻：总电阻等于各串联电阻之和，即R串 = R₁ + R₂（串联电阻越多，总电阻越大）。
并联电阻：总电阻的倒数等于各并联电阻的倒数之和，即1/R并 = 1/R₁ + 1/R₂（变形：R并=R₁R₂/(R₁+R₂)，并联电阻越多，总电阻越小）。
串、并联电阻规律：串联分压（U₁/U₂=R₁/R₂），并联分流（I₁/I₂=R₂/R₁）。
3. 欧姆定律的应用
（1）伏安法测电阻
原理：R = U/I。
器材：电源、开关、导线、待测电阻、电流表、电压表、滑动变阻器。
电路图：电流表与待测电阻串联，电压表与待测电阻并联，滑动变阻器串联在电路中。
步骤：①按电路图连接电路（开关断开，滑动变阻器调到最大阻值）；②闭合开关，调节滑动变阻器，记录几组U、I数据；③计算每组的R，求平均值（减小误差）。
（2）电路动态分析
串联电路：滑动变阻器阻值增大→总电阻增大→总电流减小→定值电阻两端电压减小→滑动变阻器两端电压增大。
并联电路：滑动变阻器阻值变化，不影响其他支路的电流和电压，只改变所在支路的电流和干路电流。
（3）短路和断路的判断
断路：电流表无示数，电压表有示数且接近电源电压（与电压表并联的部分断路）。
短路：电流表有示数，电压表无示数（与电压表并联的部分短路）。

七、电功率
1. 电功
定义：电流所做的功，实质是电能转化为其他形式能的过程，电流做功多少等于消耗电能的多少。
公式：W = UIt（通用公式），变形公式：W=I²Rt=U²t/R（纯电阻电路）。
单位：焦耳（J），常用单位：千瓦时（kW·h，度），1kW·h=3.6×10⁶J。
测量：电能表（测电路消耗的电能），读数：最后一位是小数，两次读数之差为这段时间消耗的电能。
2. 电功率
定义：电流在单位时间内所做的功，表示电流做功的快慢。
公式：P = W/t = UI（通用公式），变形公式：P=I²R=U²/R（纯电阻电路）。
单位：瓦特（W），常用单位：千瓦（kW），1kW=10³W。
额定电压和额定功率：①额定电压（U额）：用电器正常工作时的电压；②额定功率（P额）：用电器在额定电压下的功率；③实际电压（U实）：用电器实际工作时的电压；④实际功率（P实）：用电器在实际电压下的功率。
注意：当U实=U额时，P实=P额，用电器正常工作；当U实<U额时，P实<P额，用电器不能正常工作；当U实>U额时，P实>P额，用电器易损坏。
3. 测量小灯泡的电功率
原理：P = UI。
器材：电源、开关、导线、小灯泡、电流表、电压表、滑动变阻器。
电路图：与伏安法测电阻电路图相同。
步骤：①连接电路（开关断开，滑动变阻器调最大阻值）；②闭合开关，调节滑动变阻器，使小灯泡两端电压为额定电压，记录I额，计算P额=U额I额；③调节滑动变阻器，使U实<U额和U实>U额，分别记录I实，计算P实；④整理器材。
注意：不能求电功率的平均值（因为不同电压下，实际功率不同）。
4. 电与热（焦耳定律）
焦耳定律：电流通过导体产生的热量，跟电流的二次方成正比，跟导体的电阻成正比，跟通电时间成正比。
公式：Q = I²Rt（通用公式），纯电阻电路中，Q=W=UIt=U²t/R=Pt。
应用：电热器（电饭锅、电熨斗、电暖器），利用电流的热效应，发热体用电阻大、熔点高的合金制成。
防止电热的危害：电视机、电脑的散热窗，电动机的散热风扇。
5. 串、并联电路的电功和电功率规律
串联电路：①电功：W₁/W₂=R₁/R₂；②电功率：P₁/P₂=R₁/R₂（电阻越大，消耗的电功、电功率越大）。
并联电路：①电功：W₁/W₂=R₂/R₁；②电功率：P₁/P₂=R₂/R₁（电阻越小，消耗的电功、电功率越大）。
总电功和总功率：W总=W₁+W₂，P总=P₁+P₂。
6. 安全用电
家庭电路的组成：进户线（火线、零线）→电能表→总开关→保险丝→用电器、插座。
火线和零线的辨别：测电笔（笔尖接触被测导线，手接触笔尾金属体，氖管发光为火线，不发光为零线）。
保险丝：用电阻率大、熔点低的铅锑合金制成，作用：电流过大时，熔化切断电路，保护电路（不能用铜丝、铁丝代替）。
三孔插座：“左零右火上接地”，接地的作用：防止用电器金属外壳漏电时触电。
触电事故：①单线触电（接触火线和大地）；②双线触电（接触火线和零线）；高压带电体不能靠近，更不能接触。
安全用电原则：①不接触低压带电体，不靠近高压带电体；②更换灯泡、搬动电器前断开电源开关；③不弄湿用电器，不损坏绝缘层；④保险装置、插座、导线、家用电器等达到使用寿命及时更换。

八、电与磁
1. 磁现象
磁体：具有磁性的物体，分为天然磁体和人造磁体，磁体两端磁性最强（磁极），中间最弱。
磁极：北极（N极）、南极（S极），同名磁极相互排斥，异名磁极相互吸引。
磁化：使原来没有磁性的物体获得磁性的过程（钢能被永久磁化，软铁只能被暂时磁化）。
磁场：磁体周围存在的一种看不见、摸不着的物质，对放入其中的磁体有力的作用。
磁场的方向：规定小磁针静止时N极所指的方向为该点的磁场方向。
磁感线：为了形象描述磁场而画出的假想曲线，磁感线从N极出发，回到S极，磁感线密的地方磁场强，疏的地方磁场弱。
地磁场：地球周围存在的磁场，地磁北极在地理南极附近，地磁南极在地理北极附近，存在磁偏角（沈括最早发现）。
2. 电生磁（电流的磁效应）
奥斯特实验：证明通电导体周围存在磁场，即电生磁，磁场方向与电流方向有关。
通电螺线管的磁场：与条形磁体的磁场相似，两端为N、S极，极性与电流方向有关，用安培定则（右手螺旋定则）判断：右手握住螺线管，四指指向电流方向，大拇指指向的一端为N极。
电磁铁：由通电螺线管和铁芯组成，铁芯被磁化后，增强磁场。
影响电磁铁磁性强弱的因素：①电流的大小（电流越大，磁性越强）；②线圈的匝数（匝数越多，磁性越强）；③有无铁芯（有铁芯磁性强）。
特点：磁性的有无由电流的通断控制，磁性的强弱由电流大小和线圈匝数控制，磁极方向由电流方向控制。
应用：电磁继电器、电铃、电磁起重机。
3. 电磁继电器和扬声器
电磁继电器：利用低电压、弱电流控制高电压、强电流的装置，实质是由电磁铁控制的开关。
扬声器：把电信号转化为声信号的装置，由永久磁体、线圈、纸盆组成，利用通电导体在磁场中受力运动的原理。
4. 磁场对电流的作用
通电导体在磁场中会受到力的作用，受力方向与电流方向和磁场方向有关（改变其中一个，受力方向改变；同时改变两个，受力方向不变）。
应用：电动机（直流电动机），原理：通电线圈在磁场中受力转动，能量转化：电能转化为机械能。
直流电动机的换向器：作用是改变线圈中的电流方向，使线圈持续转动。
5. 磁生电（电磁感应）
电磁感应现象：闭合电路的一部分导体在磁场中做切割磁感线运动时，导体中产生感应电流，即磁生电，发现者：法拉第。
感应电流的方向：与导体切割磁感线的方向和磁场方向有关（改变其中一个，感应电流方向改变；同时改变两个，感应电流方向不变）。
产生感应电流的条件：①电路闭合；②一部分导体做切割磁感线运动。
应用：发电机，原理：电磁感应现象，能量转化：机械能转化为电能。
交流电（AC）：大小和方向随时间周期性变化的电流，我国电网的交流电频率为50Hz，周期0.02s，电流方向每秒改变100次。
直流电（DC）：大小和方向不随时间变化的电流。

九、力学基础（运动和力）
1. 长度和时间的测量
长度的单位：米（m），常用单位：千米（km）、分米（dm）、厘米（cm）、毫米（mm）、微米（μm）、纳米（nm），1km=10³m，1m=10dm=10²cm=10³mm=10⁶μm=10⁹nm。
长度的测量工具：刻度尺，使用前看量程、分度值、零刻度线；测量时尺要放正，刻度线紧贴被测物体；读数时视线与尺面垂直，估读到分度值的下一位；记录结果包括数字和单位。
误差：测量值与真实值之间的差异，不可避免，只能减小（方法：多次测量求平均值、选用精密仪器、改进测量方法），错误可以避免。
时间的单位：秒（s），常用单位：分钟（min）、小时（h），1h=60min=3600s。
时间的测量工具：停表、钟表。
2. 运动的描述
机械运动：物体位置的变化，是宇宙中最普遍的运动形式。
参照物：判断物体运动或静止时，被选作标准的物体，参照物可以任意选择，选择不同的参照物，物体的运动状态可能不同（运动和静止的相对性）。
匀速直线运动：物体沿直线且速度不变的运动，是最简单的机械运动。
变速运动：物体速度大小或方向发生变化的运动，日常所见的运动多为变速运动。
3. 运动的快慢（速度）
速度：表示物体运动快慢的物理量，定义：路程与时间的比值。
公式：v = s/t（变形公式：s=vt、t=s/v）。
单位：米每秒（m/s），常用单位：千米每小时（km/h），1m/s=3.6km/h。
平均速度：变速运动中，用总路程除以总时间表示物体的平均快慢，公式：v平=s总/t总。
4. 力
定义：力是物体对物体的作用，力不能脱离物体而存在，施力物体和受力物体同时存在。
单位：牛顿（N），1N的力约等于托起两个鸡蛋的力。
力的作用效果：①改变物体的形状（如捏橡皮泥）；②改变物体的运动状态（运动方向、速度大小改变）。
力的三要素：大小、方向、作用点，三要素都能影响力的作用效果。
力的示意图：用一根带箭头的线段表示力，线段的长度表示力的大小，箭头表示力的方向，起点或终点表示力的作用点。
力的相互性：物体间力的作用是相互的，施力物体同时也是受力物体，实例：划船、推墙时人后退。
5. 弹力
弹性：物体受力发生形变，撤去力后恢复原状的性质；塑性：物体受力发生形变，撤去力后不能恢复原状的性质。
弹力：物体由于发生弹性形变而产生的力，实例：压力、支持力、拉力、弹簧的弹力。
弹簧测力计：测量力的大小的工具，原理：在弹性限度内，弹簧的伸长量与受到的拉力成正比。
弹簧测力计使用：①调零；②不能超过量程；③拉力方向与弹簧轴线方向一致；④读数时视线与刻度面垂直。
6. 重力
定义：由于地球的吸引而使物体受到的力，符号G，施力物体是地球。
大小：物体所受的重力与它的质量成正比，公式：G = mg（g=9.8N/kg，粗略计算取10N/kg，含义：质量为1kg的物体受到的重力为9.8N）。
方向：竖直向下，应用：重垂线、水平仪（检查物体是否水平或竖直）。
重心：重力的作用点，形状规则、质量分布均匀的物体，重心在其几何中心；不规则物体的重心可用悬挂法确定，重心不一定在物体上。
7. 摩擦力
定义：两个相互接触的物体，当它们发生相对运动或有相对运动趋势时，在接触面上产生的阻碍相对运动的力。
分类：①静摩擦力（有相对运动趋势，未发生相对运动）；②滑动摩擦力（发生相对滑动）；③滚动摩擦力（发生相对滚动，滚动摩擦力远小于滑动摩擦力）。
影响滑动摩擦力大小的因素：①接触面的粗糙程度（越粗糙，摩擦力越大）；②压力的大小（压力越大，摩擦力越大），与接触面积、运动速度无关。
增大有益摩擦的方法：①增大压力；②增大接触面的粗糙程度；③变滚动为滑动。
减小有害摩擦的方法：①减小压力；②减小接触面的粗糙程度；③变滑动为滚动；④使接触面彼此分离（如加润滑油、气垫船、磁悬浮列车）。
8. 二力平衡
平衡状态：物体保持静止状态或匀速直线运动状态。
二力平衡：物体在两个力的作用下，处于平衡状态，这两个力相互平衡。
二力平衡的条件：两个力大小相等、方向相反、作用在同一直线上、作用在同一物体上（同体、等大、反向、共线）。
平衡力和相互作用力的区别：平衡力作用在同一物体上，相互作用力作用在两个不同物体上。
9. 牛顿第一定律（惯性定律）
阻力对物体运动的影响：平面越光滑，小车运动的距离越远，说明小车受到的阻力越小，速度减小得越慢。
牛顿第一定律：一切物体在没有受到力的作用时，总保持静止状态或匀速直线运动状态（实验推理法）。
惯性：物体保持原来运动状态不变的性质，一切物体都有惯性，惯性是物体的固有属性，只与质量有关（质量越大，惯性越大），与速度无关。
惯性的利用和防止：①利用：跳远助跑、拍打衣服上的灰尘、扔东西；②防止：汽车刹车时系安全带、汽车安装防撞梁、限速行驶。
10. 力与运动的关系
物体不受力或受平衡力时，运动状态不变（静止或匀速直线运动）。
物体受非平衡力时，运动状态改变（速度大小或方向改变）。

十、压强和浮力
1. 压强
（1）压力
定义：垂直作用在物体表面上的力，方向：垂直于接触面指向被压物体，大小：不一定等于重力（水平面上F=G）。
（2）压强
定义：物体单位面积上受到的压力，表示压力的作用效果。
公式：p = F/S（变形公式：F=pS、S=F/p）。
单位：帕斯卡（Pa），1Pa=1N/m²，1Pa的压强约等于把一张纸平铺在桌面上的压强。
增大压强的方法：①增大压力；②减小受力面积（如磨刀、钉尖做细）。
减小压强的方法：①减小压力；②增大受力面积（如书包带做宽、坦克装履带）。
（3）液体的压强
液体压强的特点：①液体对容器底和壁都有压强；②液体内部向各个方向都有压强；③液体的压强随深度的增加而增大；④同一深度，液体向各个方向的压强相等；⑤不同液体的压强与密度有关（密度越大，压强越大）。
液体压强的公式：p = ρgh（ρ→液体密度，单位kg/m³；g→9.8N/kg；h→深度，单位m，指从液面到该点的竖直距离）。
连通器：上端开口、底部相连通的容器，原理：连通器中装同种液体，静止时各液面保持相平，应用：茶壶、船闸、锅炉水位计、过路涵洞。
（4）大气压强
大气压的存在：马德堡半球实验证明了大气压的存在，实例：覆杯实验、吸管吸饮料、钢笔吸墨水。
大气压的测量：托里拆利实验，标准大气压下，水银柱的高度为760mm，标准大气压p₀=1.013×10⁵Pa。
大气压的变化：①随高度的增加而减小（海拔越高，气压越低）；②晴天的气压比阴天高，冬天的气压比夏天高。
液体的沸点与气压的关系：气压越大，液体的沸点越高（应用：高压锅）；气压越小，沸点越低（高山上煮不熟饭）。
大气压的应用：活塞式抽水机、离心式水泵。
2. 浮力
（1）浮力的定义和产生原因
浮力：浸在液体（或气体）中的物体受到液体（或气体）向上的托力，方向：竖直向上。
产生原因：液体（或气体）对物体向上和向下的压力差，F浮=F向上-F向下。
（2）阿基米德原理
内容：浸在液体中的物体受到的浮力，大小等于它排开的液体所受的重力。
公式：F浮 = G排 = ρ液gV排（ρ液→液体密度，kg/m³；V排→物体排开液体的体积，m³，等于物体浸入液体的体积）。
适用范围：液体和气体。
（3）物体的浮沉条件（浸没在液体中）
浮沉条件（比较浮力和重力）：
①F浮 > G → 上浮，最终漂浮（F浮'=G）；
②F浮 = G → 悬浮（可以停在液体任何深度）；
③F浮 < G → 下沉，最终沉底。
浮沉条件（比较物体密度和液体密度，实心物体）：
①ρ物 < ρ液 → 上浮，最终漂浮；
②ρ物 = ρ液 → 悬浮；
③ρ物 > ρ液 → 下沉。
（4）浮力的应用
轮船：利用空心法，增大排开液体的体积，从而增大浮力，漂浮时F浮=G船，排水量：轮船满载时排开液体的质量。
潜水艇：通过改变自身的重力实现上浮和下沉（浮力不变）。
气球和飞艇：利用空气的浮力，内充密度小于空气的气体（氢气、氦气、热空气），通过改变气囊体积改变浮力，实现升降。
密度计：测量液体密度的工具，漂浮在液体中，F浮=G，刻度上小下大，上疏下密。

十一、功和机械能
1. 功
定义：力作用在物体上，且物体在力的方向上移动了一段距离，这个力就对物体做了功。
两个必要因素：①作用在物体上的力；②物体在力的方向上通过的距离。
不做功的三种情况：①有力无距离（如推而未动）；②有距离无力（如物体由于惯性运动）；③力和距离垂直（如提水水平走）。
公式：W = Fs（F→力，N；s→距离，m）。
单位：焦耳（J），1J=1N·m，1J的功约等于把两个鸡蛋举高1m做的功。
2. 功率
定义：单位时间内所做的功，表示做功的快慢。
公式：P = W/t = Fv（v→物体运动的速度，m/s）。
单位：瓦特（W），常用单位：千瓦（kW），1kW=10³W，1W=1J/s。
3. 机械能
（1）动能
定义：物体由于运动而具有的能。
影响因素：①质量（质量越大，动能越大）；②速度（速度越大，动能越大，速度对动能的影响更大）。
（2）势能
重力势能：物体由于被举高而具有的能，影响因素：①质量（质量越大，重力势能越大）；②高度（高度越高，重力势能越大）。
弹性势能：物体由于发生弹性形变而具有的能，影响因素：弹性形变的大小（形变越大，弹性势能越大）。
（3）机械能
定义：动能和势能的总和，单位：焦耳（J）。
机械能的转化：动能和重力势能、弹性势能之间可以相互转化，实例：滚摆、单摆、瀑布（重力势能转化为动能）、撑杆跳高（弹性势能转化为重力势能和动能）。
机械能的守恒：在只有重力或弹力做功的情况下，机械能的总量保持不变（实际中由于摩擦等阻力，机械能会转化为内能，总量减小）。
4. 功的原理和机械效率
（1）简单机械
杠杆：①定义：一根硬棒，在力的作用下能绕着固定点支点（O）转动；②五要素：支点、动力（F₁）、阻力（F₂）、动力臂（L₁，支点到动力作用线的距离）、阻力臂（L₂，支点到阻力作用线的距离）；③平衡条件：F₁L₁ = F₂L₂；④分类：省力杠杆（L₁>L₂，省力费距离，如撬棍、扳手）、费力杠杆（L₁<L₂，费力省距离，如镊子、筷子）、等臂杠杆（L₁=L₂，不省力不费力，如天平、定滑轮）。
滑轮：①定滑轮：轴固定不动，实质是等臂杠杆，不省力，能改变力的方向；②动滑轮：轴随物体一起运动，实质是省力杠杆，能省一半力，不能改变力的方向，费距离；③滑轮组：由定滑轮和动滑轮组成，既能省力，又能改变力的方向，省力情况：承担物重的绳子段数为n，拉力F=G物/n（不计动滑轮重、绳重和摩擦），F=(G物+G动)/n（计动滑轮重，不计绳重和摩擦）。
斜面：省力机械，斜面越缓，越省力，费距离，公式：FL=Gh（F→拉力，L→斜面长，G→物重，h→斜面高）。
（2）功的原理
内容：使用任何机械都不省功，即使用机械时，人们所做的功，都等于不用机械时直接对物体做的功（实际中由于额外功，使用机械所做的功大于直接做的功）。
（3）机械效率
有用功（W有）：对人们有用的功，如提升物体时，W有=Gh。
额外功（W额）：对人们无用但不得不做的功，如克服动滑轮重、绳重、摩擦做的功。
总功（W总）：有用功和额外功的总和，W总=W有+W额=Fs（滑轮组中s=nh）。
机械效率（η）：有用功跟总功的比值，公式：η = W有/W总 × 100%。
特点：机械效率总小于1（因为额外功不可避免），无单位，用百分数表示。
提高机械效率的方法：①减小额外功（减轻动滑轮重、加润滑油减小摩擦、减轻绳重）；②增大有用功（增加提升的物重）。

十二、热学（内能）
1. 分子动理论
物质是由分子和原子组成的，分子很小，直径约10⁻¹⁰m。
分子在永不停息地做无规则运动，实例：扩散现象（气体、液体、固体都能发生扩散，温度越高，扩散越快）、闻到花香。
分子间存在相互作用的引力和斥力（距离较小时，斥力起主要作用；距离较大时，引力起主要作用；距离很远时，作用力可以忽略）。
2. 内能
定义：物体内部所有分子做无规则运动的动能和分子势能的总和，一切物体都有内能，内能的单位：焦耳（J）。
影响内能大小的因素：①温度（同一物体，温度越高，内能越大）；②质量（质量越大，分子数越多，内能越大）；③状态（如冰熔化为水，内能增大）；④物质的种类。
内能的改变方式：
①做功：外界对物体做功，物体内能增加；物体对外界做功，物体内能减少（实例：压缩空气引火、搓手取暖、水蒸气顶起壶盖），能量转化：其他形式的能与内能相互转化。
②热传递：高温物体向低温物体传递热量，物体吸收热量，内能增加；放出热量，内能减少，条件：存在温度差，终止条件：温度相同，能量转移：内能从一个物体转移到另一个物体，形式不变。
热量（Q）：在热传递过程中，传递能量的多少，单位：焦耳（J），不能说“物体含有热量”，只能说“吸收”或“放出”热量。
3. 比热容
定义：单位质量的某种物质，温度升高（或降低）1℃所吸收（或放出）的热量，符号c，是物质的固有属性，与质量、温度、吸放热多少无关，只与物质的种类和状态有关。
公式：Q吸 = cm(t - t₀)、Q放 = cm(t₀ - t)（c→比热容，J/(kg·℃)；m→质量，kg；t₀→初温，℃；t→末温，℃）。
单位：焦耳每千克摄氏度（J/(kg·℃)）。
水的比热容：c水=4.2×10³J/(kg·℃)，较大，应用：做冷却剂、取暖、调节气候。
比热容的应用：质量相同的不同物质，吸收相同的热量，比热容小的温度升高得快（如沙子和水）。
4. 热机
热机：把内能转化为机械能的装置，常见的热机：蒸汽机、内燃机（汽油机、柴油机）、汽轮机、喷气发动机。
内燃机的工作过程：四个冲程，分别为吸气冲程、压缩冲程、做功冲程、排气冲程，一个工作循环中，活塞往复2次，曲轴转动2周，对外做功1次。
压缩冲程：机械能转化为内能（活塞压缩气体做功）。
做功冲程：内能转化为机械能（燃气推动活塞做功，是唯一对外做功的冲程）。
热值（q）：单位质量（或体积）的燃料完全燃烧所放出的热量，是燃料的固有属性，只与燃料的种类有关。
固体、液体燃料热值公式：Q放 = mq（m→质量，kg；q→热值，J/kg）。
气体燃料热值公式：Q放 = Vq（V→体积，m³；q→热值，J/m³）。
热机的效率（η）：热机用来做有用功的那部分能量与燃料完全燃烧放出的能量之比，公式：η = W有/Q放 × 100%。
提高热机效率的方法：①使燃料充分燃烧；②减小机械摩擦和散热；③利用废气的内能。
热机对环境的影响：①大气污染（排放废气，如一氧化碳、二氧化硫）；②噪声污染，防治：改进热机、安装尾气净化装置、使用清洁能源。
5. 能量的转化和守恒定律
能量的转化：各种形式的能之间可以相互转化，实例：摩擦生热（机械能转化为内能）、发电机（机械能转化为电能）、电动机（电能转化为机械能）、光合作用（光能转化为化学能）。
能量守恒定律：能量既不会凭空消灭，也不会凭空产生，它只会从一种形式转化为其他形式，或者从一个物体转移到另一个物体，而在转化和转移的过程中，能量的总量保持不变，是自然界最普遍、最重要的定律之一。

十三、能源与可持续发展
1. 能源的分类
按产生方式分：①一次能源（直接从自然界获取，如化石能源、风能、水能、太阳能、核能）；②二次能源（由一次能源转化而来，如电能、汽油、柴油、煤气）。
按是否可再生分：①可再生能源（可以从自然界源源不断获取，如风能、水能、太阳能、生物质能）；②不可再生能源（短期内不能从自然界得到补充，如化石能源（煤、石油、天然气）、核能）。
化石能源：千百万年前埋在地下的动植物经过漫长的地质年代形成的，属于不可再生能源。
2. 核能
原子的结构：原子由原子核和核外电子组成，原子核由质子和中子组成。
核能：原子核分裂或聚合时释放的能量，又称原子能。
核裂变：重原子核分裂成几个较轻的原子核，释放出核能，应用：核电站、原子弹（不加控制的核裂变）。
核聚变：轻原子核结合成较重的原子核，释放出更大的核能，应用：氢弹（不加控制的核聚变），目前无法人工控制核聚变。
核电站：利用核裂变发电，核心设备是核反应堆，能量转化：核能→内能→机械能→电能。
3. 太阳能
太阳能的来源：太阳内部发生核聚变，释放出巨大的能量。
太阳能的优点：①清洁、无污染；②取之不尽、用之不竭；③分布广泛，无需运输。
太阳能的利用方式：①光热转换（如太阳能热水器、太阳能灶）；②光电转换（如太阳能电池，将太阳能转化为电能）；③光化转换（如光合作用，将太阳能转化为化学能）。
4. 能源与可持续发展
能源危机：随着人类对能源的需求不断增加，不可再生能源日益枯竭，出现能源危机。
能源消耗对环境的影响：①化石能源燃烧产生的废气造成大气污染、酸雨、温室效应；②核废料的放射性污染。
可持续发展：既要满足当代人的需求，又不损害后代人满足其需求的能力。
解决能源问题的途径：①节约能源，提高能源的利用率；②开发和利用新能源（太阳能、风能、水能、地热能、潮汐能、生物质能、核能等）；③发展绿色能源，减少对环境的污染。
未来的理想能源应具备的条件：①储量丰富；②价格便宜；③技术成熟；④安全、清洁，不污染环境。

十四、物理实验基础与操作
物理实验基本要求
实验原则：遵循单一变量原则、对照原则、多次实验原则，多次实验分为寻找普遍规律、减小实验误差两种目的。
实验器材使用：测量类器材先校零、看量程和分度值；操作类器材严格遵循使用规则，避免损坏。
实验数据处理：如实记录数据，不随意修改；数据处理可采用计算平均值、绘制图像、分析趋势等方法；误差分为系统误差和偶然误差，可通过改进装置、多次测量减小偶然误差。
实验报告书写：包含实验名称、实验目的、实验器材、实验步骤、实验数据、实验结论、误差分析七部分。
1.基础测量实验
（1）长度和时间的测量
器材：刻度尺、停表、三角板（辅助测量）。
步骤：①刻度尺校零，将刻度尺紧贴被测物体，用三角板辅助保证垂直测量；②读数时视线与尺面垂直，估读到分度值下一位；③停表使用时，先按启动键，结束按停止键，读数时注意小表盘和大表盘的示数配合。
误差分析：刻度尺刻度磨损、视线偏折会造成误差；多次测量求平均值减小误差。
（2）质量和密度的测量
质量测量（天平）：①天平放在水平台上，游码移至零刻度线，调节平衡螺母使横梁平衡；②左物右码，用镊子夹取砝码，轻拿轻放，必要时移动游码；③物体质量 = 砝码质量 + 游码示数。
密度测量（固体 / 液体）：
固体密度（排水法）：①用天平测固体质量 m；②量筒中装适量水，记录体积 V₁；③用细线系住固体，缓慢浸入量筒水中，记录总体积 V₂；④密度 ρ=m/(V₂-V₁)。
液体密度（差值法）：①用天平测烧杯和液体总质量 m₁；②将部分液体倒入量筒，记录体积 V；③用天平测烧杯和剩余液体质量 m₂；④密度 ρ=(m₁-m₂)/V。
注意：测量易吸水固体（如木块）需用排油法或覆膜法；测量腐蚀性液体需用专用容器。
2.力学核心实验
（1）探究影响滑动摩擦力大小的因素
器材：木块、弹簧测力计、木板、毛巾、砝码。
步骤：①将木块放在木板上，用弹簧测力计水平匀速拉动，记录示数 F₁（二力平衡，F=f）；②将毛巾铺在木板上，重复上述步骤，记录示数 F₂；③在木块上加砝码，放在木板上，匀速拉动，记录示数 F₃。
结论：滑动摩擦力与接触面粗糙程度和压力大小有关，接触面越粗糙、压力越大，滑动摩擦力越大。
（2）探究浮力大小与哪些因素有关
器材：弹簧测力计、铁块、烧杯、水、盐水。
步骤：①用测力计测铁块重力 G；②将铁块部分浸入水中，测拉力 F₁，计算浮力 F 浮₁=G-F₁；③将铁块完全浸入水中，测拉力 F₂，计算浮力 F 浮₂=G-F₂；④将铁块完全浸入盐水中，测拉力 F₃，计算浮力 F 浮₃=G-F₃。
结论：浮力大小与液体密度和物体排开液体的体积有关，液体密度越大、排开体积越大，浮力越大；与物体浸入深度无关。
（3）探究杠杆的平衡条件
器材：杠杆、支架、钩码、弹簧测力计。
步骤：①调节杠杆两端平衡螺母，使杠杆在水平位置平衡（便于测量力臂）；②在杠杆两侧挂不同数量钩码，调节钩码位置，使杠杆再次水平平衡，记录动力 F₁、动力臂 L₁、阻力 F₂、阻力臂 L₂；③改变钩码数量和位置，多次实验，记录数据。
结论：杠杆平衡条件为F₁L₁=F₂L₂（动力 × 动力臂 = 阻力 × 阻力臂）。
3.电学核心实验
（1）探究串、并联电路的电流 / 电压规律
器材：电源、开关、导线、电流表、电压表、两个小灯泡。
串联电路步骤：①将电源、开关、两个小灯泡、电流表串联，闭合开关，测电路中不同位置电流；②将电压表分别并联在小灯泡两端、电源两端，测各部分电压。
并联电路步骤：①将两个小灯泡并联，电流表分别接在干路、各支路，测电流；②电压表并联在电源和各小灯泡两端，测电压。
结论：串联电流处处相等、总电压等于各部分电压之和；并联干路电流等于支路电流之和、各支路电压等于电源电压。
（2）探究电流与电压、电阻的关系
器材：电源、开关、导线、电流表、电压表、定值电阻、滑动变阻器。
探究电流与电压的关系：①保持定值电阻 R 不变，调节滑动变阻器，改变电阻两端电压，记录多组 U、I 数据；②绘制 I-U 图像，分析趋势。
探究电流与电阻的关系：①更换不同阻值的定值电阻，调节滑动变阻器，保持电阻两端电压不变，记录多组 R、I 数据；②绘制 I-R 图像，分析趋势。
结论：电阻一定时，电流与电压成正比；电压一定时，电流与电阻成反比。
4.热学基础实验
（1）探究晶体熔化和凝固的特点
器材：海波（晶体）、蜡（非晶体）、酒精灯、烧杯、试管、温度计、石棉网、停表。
步骤：①将海波装入试管，放入盛水的烧杯中（水浴加热，使受热均匀）；②用温度计测海波温度，每隔 1min 记录一次温度和状态；③海波熔化后，停止加热，继续记录温度和状态，直至凝固。
结论：晶体熔化时吸热，温度保持不变（有熔点）；凝固时放热，温度保持不变（有凝固点）；非晶体熔化和凝固时温度持续变化。
（2）探究水的沸腾特点
器材：烧杯、水、酒精灯、石棉网、温度计、铁架台、停表。
步骤：①将水倒入烧杯，温度计玻璃泡浸入水中（不接触杯底和杯壁）；②用酒精灯加热，每隔 1min 记录一次温度；③水沸腾后，继续加热，记录温度变化。
结论：水沸腾时吸热，温度保持不变（沸点）；标准大气压下，水的沸点为 100℃；沸点随气压升高而升高。

十五、物理现象与生活应用
1.声现象的生活应用
回声定位：声呐探测海底深度、鱼群位置；蝙蝠利用回声辨别障碍物，公式：s=vt/2（t 为发射到接收回声的总时间）。
声音的特性应用：医生用听诊器减小声音分散，增大响度；不同乐器靠音色区分；超声波洗牙利用声音的能量，B 超利用声音传递信息。
噪声控制：城市道路旁的隔音板（传播过程中减弱）；摩托车的消声器（声源处减弱）；防噪声耳塞（人耳处减弱）。
2.光现象的生活应用
光的直线传播：皮影戏、小孔成像、激光准直、排队看齐；树荫下的光斑是太阳的倒立实像。
光的反射：穿衣镜、汽车后视镜（凸面镜，扩大视野）、潜望镜；黑板反光是镜面反射，漫反射使我们能从各个方向看到物体。
光的折射：放大镜看物体、照相机拍照、投影仪成像；渔民叉鱼时瞄准鱼的下方（光的折射使鱼的像变浅）；海市蜃楼是光的折射和全反射现象。
看不见的光：红外线烤箱、红外测温仪；紫外线消毒灯、验钞机；防晒衣阻挡紫外线，防止皮肤受伤。
3.物态变化的生活应用
熔化与凝固：雪糕熔化吸热降温；北方冬天菜窖放几桶水，水凝固放热防止菜冻坏；保险丝利用低熔点金属熔化切断电路。
汽化与液化：夏天洒水降温（水蒸发吸热）；吹风机吹干头发（加快空气流速、提高温度，加快蒸发）；冰箱制冷利用制冷剂的汽化吸热和液化放热；水壶口的 “白气” 是水蒸气液化形成的小水珠。
升华与凝华：干冰人工降雨（干冰升华吸热，使水蒸气凝华成小冰晶）；樟脑丸变小、冰冻的衣服变干是升华现象；霜、雪、雾凇是凝华现象。
4.力学的生活应用
重力与弹力：重垂线检查墙壁是否竖直；弹簧测力计测物重；跳高运动员利用撑杆的弹性势能转化为重力势能。
摩擦力：鞋底、轮胎的花纹增大接触面粗糙程度，增大摩擦；行李箱的轮子变滑动为滚动，减小摩擦；自行车刹车时用力捏闸，增大压力，增大摩擦。
压强：菜刀磨得很薄，减小受力面积增大压强；书包带做宽，增大受力面积减小压强；高压锅增大锅内气压，提高水的沸点，使食物易熟；吸盘挂钩利用大气压强固定在墙上。
浮力：轮船利用空心法增大排液体积，漂浮在水面；潜水艇通过注水和排水改变自身重力，实现浮沉；热气球加热内部空气，减小密度，利用浮力升空；密度计测液体密度，漂浮时浮力等于重力。
功和机械能：盘山公路相当于斜面，省力费距离；举重运动员将杠铃举起，对杠铃做功；水电站利用水的重力势能转化为电能；蹦床运动利用弹性势能和动能的相互转化。
5.电学的生活应用
串并联电路：家庭电路中用电器并联，开关与用电器串联；节日小彩灯串联（部分彩灯有分压电阻）；冰箱、空调等大功率用电器单独接保险丝，防止总电流过大。
欧姆定律应用：测电笔利用分压原理，使人体两端电压低于安全电压；滑动变阻器调节电路电流，如台灯的调光旋钮、收音机的音量调节。
电功率与电热：电水壶、电饭锅利用电流的热效应；电视机、电脑的散热窗防止电热危害；家庭电路中，总功率过大（P 总 = UI 总）会导致电流过大，引发火灾。
电与磁：电铃、电磁起重机利用电磁铁；电动机应用在电风扇、洗衣机、电动车中；发电机应用在水电站、火电站，将机械能转化为电能；动圈式话筒利用电磁感应现象，将声信号转化为电信号。
6.热学与能源的生活应用
内能的利用：热水袋取暖（热传递）；搓手取暖（做功改变内能）；汽车发动机用水做冷却剂（水的比热容大）。
热机的应用：汽油机应用在小汽车、摩托车；柴油机应用在拖拉机、轮船、大型货车；热机的排气冲程排出废气，污染环境，需安装尾气净化装置。
能源的利用：家庭中使用的电能属于二次能源；太阳能热水器利用光热转换，太阳能路灯利用光电转换；农村的沼气池利用生物质能，属于可再生能源；核电站利用核能发电，核废料需妥善处理，防止放射性污染。
7.物理知识解释生活现象
夏天从冰箱拿出的饮料瓶 “出汗”：空气中的水蒸气遇冷液化成小水珠。
冬天窗户玻璃内侧有冰花：室内的水蒸气遇冷凝华成冰花。
汽车刹车后仍会向前运动：汽车具有惯性，保持原来的运动状态。
洗澡时浴室镜子变模糊，擦干后又清晰：水蒸气液化成小水珠使镜子模糊，水珠蒸发后镜子清晰。
用吸管喝饮料：大气压将饮料压入吸管，进入口中。
高压锅煮饭快：锅内气压大，水的沸点高，煮饭时间缩短。
路灯下的影子长短变化：光的直线传播，人离路灯越近，影子越短；离路灯越远，影子越长。
